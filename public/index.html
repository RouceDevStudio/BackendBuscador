⚡ Activar Ollama local

Con Ollama corriendo en tu dispositivo, NEXUS tiene un motor de lenguaje local de respaldo cuando Groq no está disponible. Son solo 3 comandos.

1

Instalar Termux (si no lo tenés)

Descargalo desde F-Droid o desde tu tienda de apps. Luego abrilo.

2

Instalar y arrancar Ollama

Copiá y pegá estos comandos en Termux, uno por uno:

pkg install ollama
OLLAMA_ORIGINS="*" ollama serve
3

Descargar el modelo (en otra ventana de Termux)

Abrí una nueva sesión de Termux (deslizá desde el borde izquierdo) y ejecutá:

ollama pull llama3.2:1b
4

Volvé a esta página y recargala

NEXUS va a detectar Ollama automáticamente y verás el indicador verde.

⚠️ El OLLAMA_ORIGINS="*" en el paso 2 es clave: permite que la página web se comunique con Ollama. Sin esto no funciona.
1

Descargá e instalá Ollama

Entrá a ollama.com, descargá el instalador y seguí los pasos (siguiente, siguiente, instalar).

2

Descargá el modelo

ollama pull llama3.2:1b

Iniciá Ollama con CORS habilitado:

OLLAMA_ORIGINS="*" ollama serve

Windows PowerShell: $env:OLLAMA_ORIGINS="*"; ollama serve

3

Recargá esta página

NEXUS detecta Ollama automáticamente. El indicador verde confirma que está activo.
